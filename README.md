## 🔹 Частина 1 — проєктування схеми

### Модель графа

```
(:User {userId, gender, age, occupation})
(:Movie {movieId, title, genres: [string]})

(:User)-[:RATED {rating, timestamp}]->(:Movie)
```

```mermaid
graph LR
    U["User\nuserId, gender, age, occupation"] -- "RATED\nrating, timestamp" --> M["Movie\nmovieId, title, genres[]"]
```

### Схема з Neo4j Browser

│[(:Movie {name: "Movie",indexes: ["movieId"],constraints: []}), (:User│[[:RATED {name: "RATED"}]]│
│ {name: "User",indexes: ["userId"],constraints: []})]                 │                          │
